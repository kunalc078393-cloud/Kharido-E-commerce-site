# Kharido E-Commerce Site - Project Report

## Project Overview
Kharido-E-commerce-site is a static e-commerce frontend built with HTML5 and CSS3. It includes a homepage, contact page, login page, signup page, and a shared stylesheet.

## Technologies Used
- HTML5 for structure, sections, navigation, product cards, and forms.
- CSS3 for theme variables, layout, typography, shadows, hover effects, focus states, and responsive media queries.
- External image URLs for hero and product visuals.
- Browser form validation through required, email, password, checkbox, textarea, and submit elements.

## Main Components
- Header and navigation: sticky dark top bar with brand and links.
- Hero: two-column welcome area with image, headline, and supporting text.
- About section: centered shop mission text.
- Product grid: responsive cards with image, title, description, and price.
- Contact form: name, email, message, and submit button.
- Login page: email/password form, remember-me option, account switch link, and guidance panel.
- Signup page: registration fields, password hint, confirmation field, and guidance panel.
- Footer: consistent copyright block.

## How It Is Built
HTML defines the content and semantic structure. CSS applies the visual design through custom properties, grid and flex layouts, card styling, form styling, transitions, and media queries.
